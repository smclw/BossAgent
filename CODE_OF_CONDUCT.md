# Code of Conduct

BossAgent welcomes contributors from different backgrounds, languages, and experience levels.

## Our Standards

Examples of positive behavior:

- Be respectful and constructive.
- Give context when reporting problems.
- Assume good intent while staying clear about facts.
- Help newcomers understand the project.

Unacceptable behavior includes:

- Harassment or personal attacks.
- Publishing private information.
- Spam, scams, or malicious links.
- Attempts to add unsafe automation without human confirmation.

## Enforcement

Maintainers may remove comments, close issues, or block participants who violate this code of conduct.

If you need to report a concern, open a private security report on GitHub or contact the maintainers through the repository.
