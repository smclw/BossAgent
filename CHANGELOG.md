# Changelog

All notable changes to BossAgent will be documented here.

## 0.1.0 - Initial Local MVP

### Available

- Streamlit local UI.
- Multi-agent orchestration with a Python fallback architecture.
- Six built-in AI employee roles.
- OpenAI-compatible model configuration.
- Experimental local GGUF model folder mode.
- PDF, DOCX, XLSX, XLS, and TXT upload parsing.
- SQLite task history.
- Markdown and DOCX report export.

### Planned

- Native CrewAI orchestration.
- More specialized agents.
- Knowledge base and long-term memory.
- External tool connectors with human confirmation.
- Team workspace and permissions.
