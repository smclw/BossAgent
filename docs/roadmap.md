# Roadmap

## V0.1 Local MVP

Status: Available

- Multi-agent collaboration.
- Task input.
- File parsing.
- Local task history.
- Markdown / DOCX report export.
- OpenAI-compatible configuration.

## V0.2 More Specialized Agents

Status: Planned

- Founder assistant.
- Sales assistant.
- Content assistant.
- Research assistant.
- Operations assistant.
- More industry-specific agents.

## V0.3 Knowledge Base and Memory

Status: Planned

- Knowledge base ingestion.
- Long-term memory.
- Better historical task management.
- Reusable business context.

## V0.4 External Tools

Status: Planned

- Browser automation.
- Email.
- Spreadsheets.
- Feishu.
- WeCom.
- Dify.
- n8n.

All external actions must require human confirmation.

## V0.5 Team Workspace

Status: Planned

- Multi-user workspace.
- Role-based permissions.
- Team task history.
- Admin controls.

## V1.0 AI Workforce OS

Status: Planned

- Deployable AI employee workspace.
- Extensible agent marketplace.
- Enterprise-ready configuration, logging, and auditing.
