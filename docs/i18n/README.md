# Internationalization

BossAgent uses multilingual README files for the project homepage.

- `README.md` is the English source version.
- Other README files should follow the English version when product claims or commands change.
- When adding a new language, update the language navigation at the top of every README.
- Community translation pull requests are welcome.

Current language files:

- `README.md`
- `README.zh-CN.md`
- `README.zh-TW.md`
- `README.ja.md`
- `README.ko.md`
- `README.es.md`
- `README.pt-BR.md`
- `README.fr.md`
