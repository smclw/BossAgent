# Translation Guide

Keep translations natural, concise, and technically accurate.

## Terms

| Term | Guidance |
| --- | --- |
| BossAgent | Do not translate. |
| AI Workforce OS | Keep the English term; local explanation is allowed. |
| AI Agent | Keep "AI Agent" when the local translation is awkward. |
| Chief of Staff Agent | Translate as a role, but keep Agent visible. |
| Opportunity Radar Agent | Translate as opportunity radar / trend radar Agent. |
| Strategy Advisor Agent | Translate as strategy advisor Agent. |
| Content Clone Agent | Translate as content clone / content avatar Agent. |
| Sales Follow-up Agent | Translate as sales follow-up Agent. |
| Document Analyst Agent | Translate as document analyst Agent. |
| local-first | Translate as local-first or local-first with explanation. |
| OpenAI-compatible API | Keep OpenAI-compatible API. |

## Rules

- Do not exaggerate unfinished features.
- Mark unfinished features as Planned or Experimental.
- Keep commands runnable.
- Keep language navigation links consistent.
- Do not copy text from other projects.
